IoTDashboard User Guide

1. Firebase Setup

1.1. Firebase Intro

Firebase is an app development platform that helps you build and grow apps and games users love. Backed by Google and trusted by millions of businesses around the world.

1.2. Create Firebase Project

1. Create a gmail account and login.

2. Navigate to Firebase console.

3. Click on "Create a project" button.

4. Give your project a name.

5. Click "Continue" button.

6. Click "Create project" button.

1.3. Create Realtime Database

1. In the navigation menu on the left, click on "Build" section, then click on "Realtime Database" option and hit the "Create Database" button in the middle of the screen.

2. Choose a location where you want your realtime database to be stored and click "Next".

3. Choose any of the two security mode and click "Enable".

4. After the realtime database is created, you want to define the security rules first of all. For that click on "Rules" tab in the "Realtime Database" page.

5. Use the below security rules then click "Publish" button.

{
  "rules": {
      "$uid": {
          ".write": "$uid === auth.uid",
          ".read": "$uid === auth.uid",
      },
      "MaintainedConnectionPoint": {
          ".write": true,
          ".read": true,
      }
  }
}

1.4. Define User Authentication Method

1. In order for the Firebase to authenticate the users trying to access the data, you have to define the authentication method. For that, in the navigation menu on the left, click on "Build" section, then click on "Authentication" option and hit the "Create Database" button in the middle of the authentication page.

2. Under "Sign-in method" tab, choose "Email/Password" as the sign-in method.

3. Enable "Email/Password" authentication method and click "Save".

1.5. Register IoTDashboard App in Firebase

1. To be able to integrate the IoTDashboard app with your Firebase project, you have to register the app in the project. For that, click on the settings icon in the navigation menu on the left and choose "Project settings".

2. Under "General" tab, click on the android icon within "Your apps" section.

3. Fill in the "Android package name" input with:

com.kandroid.iotdashboard

4. Click "Next".

5. Click "Next".

6. Click "Continue to console".

7. Now you are ready to use the IoTDashboard app.

2. IoTDashboard App Setup

2.1. Connect IoTDashboard App to Firebase

1. In order to connect the IoTDashboard app with your Firebase project you need to include a configuration file named google-services.json file into your app. To do that, login to your Firebase console from your android phone, then go to your project settings.

2. Scroll down to "Your apps" section and download "google-services.json" file.

3. Open the IoTDashboard app and click on the menu icon on the top left then click on "Connect to Firebase" option.

4. On the "Firebase Connection" screen, click on browse.

5. Select the "google-services.json" file you just downloaded.

6. Click on "Connect" button.

7. If the connection with Firebase is established successfully, you should see the Firebase connection status led on the top right turning into green.

2.2. Create an Account and Login

1. The final step before you can start using the IoTDashboard app, is to create a user account and login to access and manage your private data securely. For that, click on the menu icon on the top left then click on "User login" option.

2. If you did not create a user account before, click on the "Sign up" link below the "Login" button and follow the process. After you sign up successfully, you will be redirected to the "Login" page to login to your account. Please note that you must confirm your email address before your first attemt to login, this is done through a link sent to you by email. If you don't find this confirmation link in your inbox it could be in the junk section.

3. Upon successful login you will be redirected to the main page and you are now ready to add some "Things"!.

2.3. Add Things and Topics

1. Now, with all the all the necessary setups done properly, you can start adding things (clusters of topics) and topics within these things. To add a new thing, click on the (add) "+" floating button on the main page.

2. A window will pop up asking for the tag and description og the added thing. You should always assign a unique tag otherwise it won't be accepted. You can optionally give the new added thing a name /or description. Then click "Ok".

3. If the new thing is properly configured, a new thing card will be added on the main page (Dashboard). To access this thing an create or monitor topics within it, click on the thing card.

4. On "Topics" page toolbar you can see the tag and the description of the thing. To add a new topic, click on the (add) "+" floating button on the bottom of the "Topics" page.

5. There are three available types of topics in the IoTDashboard: "Analog", "Digital" and "Text". To add an "Analog" Topic, select the corresponding radio button and give the topic a unique tag and a description (optional), then click "Ok".

6. . The same way you can add a "Digital" topic with the "Digital" radio button selected instead of the "Analog".

7. . You can see the added topics listed on the "Topics" page related to the thing you are working with.
